Moltonfは人狼BBS の過去ログ観戦ビューアです。

特徴

参加者の視点で観戦

次の日へ移動するボタンを押すまで、それより先の日は表示されません。特に終盤、これで終わるのかそれとも続くのかわからないので、リアルタイムに参加しているようなワクワク感を味わえます。

あらかじめ選択した役職（ただの村人・占い師・霊能者・人狼など）に応じてキャラクターが決定され、そのキャラクターの視点で見える情報だけが表示されます。例えば、キャラクターが人狼であれば仲間たちのささやき（赤ログ）が表示されますが、村人なら表示されません。同じように能力者であれば、占い師なら占い結果といったその能力に応じた情報を知ることができます。ネタバレを避けるかたちで実際にそのキャラクターで参加しているような気分で観戦できます。

ブラウザ内で動作

お使いのブラウザ内のJavaScriptで動作します。選んだ過去ログや観戦状況はブラウザが管理するローカルコンピューター上のストレージに記憶されます。データはサーバーにはアップロードされません。利用に当たってのユーザー登録なども不要です。

過去ログはXMLファイル

過去ログとして共通アーカイブ （※2023/06/28時点ではサイトに接続できません）形式のXMLファイルを使います。あらかじめ観戦したい過去ログを入手しておいていただく必要があります。

過去ログデータの入手方法

観戦には共通アーカイブ形式のXMLファイルが必要になります。あらかじめ誰かからデータをもらうなり、自分で作成するなりして入手しておいてください。

人狼BBS用の専用クライアント「Jindolf 」の作者であり、共通アーカイブ形式の提唱者でもある olyutorskii さんが開発された「JinArchiver」を使うと、人狼BBSから指定した村データをダウンロードして共通アーカイブ形式で保存できます。

  • JinArchiver

上記のサイトから jinarchiver-X.X.X.jar（X.X.Xはバージョン番号）をダウンロードして、Javaランタイムが存在するコマンドプロンプトで実行します。

G国500番の過去ログXMLファイルをdataディレクトリに保存する実行例

java -jar jinarchiver-X.X.X.jar -land wolfg -vid 500 -outdir data

※olyutorskii さんはMoltonfの製作に関わっていません。Moltonfに関する問い合わせを olyutorskii さんに行うのはやめてください。
